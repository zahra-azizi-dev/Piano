# 🎹 Interactive Web Piano

An interactive browser-based piano built using **HTML, TailwindCSS, and Vanilla JavaScript**.

Play music with your **mouse 🖱️** or your **keyboard ⌨️**, adjust the volume in real time 🎚️👀.

Demo:(https://zahra-azizi-dev.github.io/Piano/)

## 🚀 Features

* 🎵 Clickable piano keys
* ⌨️ Full keyboard support
* 🎚️ Real-time volume control
* 👀 Show / Hide key labels
* 💫 Smooth key press animation
* ⚡ Clean and reusable JavaScript logic
* 🎨 Styled with TailwindCSS

---

## 🧠 Project Highlights

This project demonstrates:

* 🔁 Reusable function architecture (`playNote`)

* 🎯 Safe keyboard event handling
